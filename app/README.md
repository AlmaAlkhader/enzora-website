# app/

This directory is a placeholder for the future **Enzora mobile companion app**.

## What will live here

A React Native / Expo mobile app that allows customers to:

- Browse the Enzora product catalogue
- Place and track orders from their phone
- Receive push notifications about order status updates (via Firebase)
- Switch between Arabic and English interfaces

## Planned stack

- Expo (React Native)
- TypeScript
- Shared API hooks from `lib/api-client-react`

## Status

Not yet started. This folder is reserved to keep the repo structure clean for when mobile development begins.
