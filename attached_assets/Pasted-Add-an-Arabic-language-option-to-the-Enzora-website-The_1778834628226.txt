Add an Arabic language option to the Enzora website.

The website should support two languages:
- English
- Arabic

Add a clear language switcher in the navbar:
- EN
- AR

When the user clicks AR:
- The public website text should change to Arabic.
- The layout direction should become RTL.
- Text should align right.
- Navbar order and section spacing should work correctly in Arabic.
- Buttons, forms, product cards, FAQ, footer, and privacy section should all translate properly.

When the user clicks EN:
- The website returns to English.
- Layout direction becomes LTR.

Important:
Do not create a separate Arabic website.
Use the same website with language switching.

Suggested implementation:
- Create a translations file, for example:
  lib/translations.ts
- Store all public website text in English and Arabic.
- Use a language context/provider or simple state stored in localStorage.
- Save the selected language in localStorage so when the user refreshes, the chosen language stays.
- Set the HTML direction dynamically:
  - dir="ltr" for English
  - dir="rtl" for Arabic
- Set lang dynamically:
  - lang="en"
  - lang="ar"

Arabic font:
Use a clean Arabic-friendly font if possible, such as:
- Cairo
- Tajawal
- Noto Sans Arabic

Do not use a font that makes Arabic look broken or crowded.
